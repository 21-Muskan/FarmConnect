# 🌾 FarmConnect – A Social Network & Smart Assistant for Farmers

> **FarmConnect** is a full-stack AgriTech platform that combines **social media**, **AI-powered farming tools**, and **financial management** to empower farmers with knowledge, connectivity, and data-driven decision making.

---

## 🚜 Why FarmConnect?

Modern farmers need more than just information — they need **community, insights, and tools** in one place.  
FarmConnect bridges the gap by offering a **farmer-centric social network** along with **smart agricultural utilities**.

## 🌟 Core Features

### 🤝 Social Networking for Farmers
- 📝 **Community Posts** – Share text & images
- 🎥 **Agri Loop (Reels)** – Short farming videos
- ❤️ **Like / Dislike System**
- 💬 **Comments & Discussions**
- 👥 **Follow / Unfollow Farmers**
- 🌐 **Multilingual Support**

---

### 👨‍🌾 Farmer-Focused Profiles
Each farmer profile is **data-rich and agriculture-specific**:

- 🌱 Land size & land type (Irrigated / Rainfed / Dry)
- 🌾 Crops grown
- 🐄 Dairy & poultry information
- 💧 Irrigation methods
- 🧪 Fertilizers & pesticides used
- 🌽 Seed brands
- 📆 Annual crop cycles
- 📊 Profile analytics (posts, followers, engagement)

---

## 🛠️ Technical Architecture

### 🔙 Backend – Django & DRF
- **Framework:** Django + Django REST Framework
- **Authentication:** Token-based (JWT ready)
- **Database:** MySQL
- **Key Models:**
  - User & Profile
  - Post & Reel
  - Like / Dislike
  - Comment
  - Follow system
  - Expense, Income, Loan, Budget

---

### 🎨 Frontend – React
- Modern React application
- Responsive (Mobile + Desktop)
- API integration with Django backend
- Image & video uploads

---
